kurl 0.0.1

a minimalist link shortener based in node.js and sqlite.

to create a short link, send a post request to this website with your terminal, and specify the password given to you by the administrator of this site in the request. for example...

curl -X POST {{INSTANCE_URL}}/shorten -H "Content-Type: application/json" -d '{"password": "HUNTER7", "name": "information", "url": "https://example.com"}'

kurl has extremely simple, non-tracking analytics that do not log location, just referrers and click count. if you have the adminstrator password, check the analytics with the following request:

curl -X POST http://localhost:3000/analytics -H "Content-Type: application/json" -d '{"password": "HUNTER7", "name": "information"}'
